# Project Docs

This folder is split into current source-of-truth docs and generated/review asset working files.

## Current Docs

- `development/main-branch-validation.md`: branch/default-branch status, root-overlay ZIP workflow, and current-state verification sequence.
- `development/phase-0-to-3-changes.md`: summary of what this cumulative patch changes through roadmap Step 3.
- `development/save-schema.md`: current clean-break save schema decision after item catalog and NPC stock overhaul.
- `development/technical-notes.md`: stack, commands, generated data, scripts, verification, and git hygiene.
- `game/roadmap.md`: current development roadmap and milestone order.
- `game/game-logic-parity.md`: implementation-level parity checklist against original game behavior.
- `game/ui-ux-brief.md`: canonical UI and visual direction.
- `systems/trading-and-stock.md`: current barter, NPC stock, restocking, diagnostics, and extension-point spec.
- `systems/profession-stock-audit.md`: profession-by-profession stock balance audit and follow-up list.
- `assets/item-icon-pipeline.md`: item icon prompt/sheet/crop pipeline.

## Generated Or Review Assets

- `systems/profession-stock-review.md`: generated by `pnpm review:stock`; do not hand-edit as a planning doc.
- `assets/icon-prompts/`: generated item icon prompt configs.
- `assets/icon-sheets/`: generated sheet images for review and cropping.
- `ui_parts/`: current UI visual reference mockups.

These folders are useful project material, but they are not prose planning docs unless explicitly listed under Current Docs.
