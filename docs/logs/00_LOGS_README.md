# Logs Folder

This folder is for generated reports, historical notes, old phase docs, temporary audits, and one-time handoffs.

These files are not part of the normal reading order.

Current source-of-truth docs are the numbered files directly under `/docs`.

Generated reports that belong here include:

- profession stock review
- item icon lock report
- item icon lock JSON
- item icon manual review CSV
- playtest/balance report
- old phase/migration notes

Do not put permanent source-of-truth docs here.
